Building
========

In order to build (or compile) the reports, you need the following software:

- Python 3
- NumPy for Python 3
- SciPy for Python 3
- Jinja2 for Python 3
- LaTeX (pdflatex) and a bunch of packages
- Make
- pdfcrop
